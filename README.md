# CSV Grid Viewer

</div>

## Development

Running `yarn tauri dev`, start a development build of the app.

```sh
yarn tauri dev
```

## Build

Running `yarn tauri build`, AppImage, deb and rpm in `src-tauri/target/release/bundle/`.(Linux)

```sh
yarn tauri build
```
